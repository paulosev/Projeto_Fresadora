Configuração

config.h

include/config.h" — Configuração Layer
Configuração central do projeto — constantes, pinos e limites.
Todos os parâmetros de hardware, limites de segurança e ganhos PID estão centralizados neste arquivo. Para ajustar qualquer parâmetro, edite apenas este arquivo.
