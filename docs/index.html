Fresadora STM32 — Documentação da API

Sistema de controle de motor universal 1800W em DC pulsado. STM32F103C8T6 (Blue Pill) com PlatformIO.

Arquitetura

CamadaArquivosResponsabilidade
Config config.h Constantes, pinos, limites e ganhos PID centralizados
HAL pwm · rpm · corrente · controle Acesso direto ao hardware (registradores, timers, ADC). Única camada que muda ao portar para outro STM32.
Control pids · autotune Algoritmos de controle puros. Não conhece pinos nem registradores.
App maquina Lógica de aplicação de alto nível. Orquestra HAL e Control pelo nome.

Fluxo de Execução

ContextoFrequênciaO que executa
TIM3 ISR2kHz — prioridade 0 lerCorrente() → calcularRPM() → processarCascata() → verificações de segurança
Hall ISRPor pulso — prioridade 1 Registra período entre pulsos do sensor Hall
loop()~95% do CPU livre executarSoftStart() · display · botões · serial
