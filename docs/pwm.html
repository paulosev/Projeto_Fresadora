pwm.h

lib/pwm.h" — HAL Layer
Saída PWM de potência via TIM1 CH1 (PA8) a 16kHz.
Configura o TIM1 em modo PWM Modo 1 a 16kHz com resolução de 4500 degraus. Compatível com o driver IR2110S e MOSFET IRFP460.

Funções

void setupHardwarePWM()
Inicializa o TIM1 para geração de PWM a 16kHz em PA8.

Configura clock, GPIO, modo PWM Modo 1, preload e Main Output Enable (MOE). Inicia com duty cycle zero (motor desligado). Deve ser chamada antes de qualquer chamada a aplicarPWM().

⚠ Atenção: TIM1 é um Advanced Timer. O bit MOE no registrador BDTR deve ser setado, caso contrário PA8 não gera sinal — falha silenciosa comum na migração do Arduino.
void aplicarPWM(float valor)
Aplica um valor de duty cycle ao TIM1 CCR1.

O valor é limitado automaticamente ao intervalo [0, MAX_PWM] antes de ser escrito no registrador de comparação.

ParâmetroDescrição
valorDuty cycle desejado (0.0 = desligado, MAX_PWM = 100%).
